처음일 경우
1. 새로운 폴더를 만듭니다
2. 새로운 폴더에 들어가 git init을 합니다.
3. gig remote add (origin) (repo addr)를 해 받는 사람을 등록합니다.

<checkpoint>
4. 새로운 파일을 만들고 작업을 합니다.(향후 파일 관련한 모든 일들)
5. git add 새로운 파일이름으로 파일을 추가합니다.
6. git commit -m "커밋 메시지"로 커밋 메시지를 등록합니다. 
7. git push (origin)master로 해당 내용을 전송합니다. 

이후 <check point> ~ 7 반복
